# 隐私政策发布说明

`index.html` 是 EggClip AppGallery 提审使用的静态隐私政策页面，不依赖脚本、字体、图片或第三方
服务，可以直接部署到支持 HTTPS 的静态网站。

发布前：

1. 确认页面内容与 `docs/PRIVACY.md` 和 `docs/PRIVACY_DATA_INVENTORY.md` 一致。
2. 使用手机和桌面浏览器检查浅色、深色模式及窄屏排版。
3. 确认公开 URL 无需账号即可访问，且不会跳转到下载页或登录页。
4. 将最终 HTTPS URL 填入 AppGallery Connect 隐私政策地址和审核说明。
5. 确认 AppGallery 应用详情页中的开发者联系方式可用。

H-REVIEW-07 会记录最终公开 URL；在此之前不要在应用代码中写入临时地址。
