# EggClip 隐私说明

EggClip 是纯局域网剪贴板同步工具。v1 不需要账号，不连接中心服务器，不使用公网中继，也不加入遥测或崩溃上报。

## 处理的数据

- 仅同步用户主动复制或通过 HarmonyOS `PasteButton` 授权读取的 `text/plain`，单条最大 256 KiB。
- 桌面端默认保存最近 50 条且最长 7 天；用户可以关闭历史或随时清空。
- HarmonyOS 只在前台发现和连接，不能在后台静默读取系统剪贴板。
- 配对会保存设备标识、公钥、可信状态、最近连接地址和加密后的同步记录。

## 数据存放与传输

- 正文只在已认证设备之间通过局域网传输，业务帧使用应用层 AEAD 加密。
- Windows 私钥和空间密钥保存在系统凭据库；HarmonyOS 密钥优先保存在 HUKS。
- 本地数据库不保存私钥、邀请秘密、原始空间密钥或裸 SHA-256 正文摘要。
- EggClip 写入 Windows 剪贴板时设置云剪贴板排除标记，避免内容上传到 Windows 云剪贴板。

## 用户控制

- 可以暂停同步、关闭自动接收/写入、关闭历史或调整保留数量与天数。
- 可以重命名或移除可信设备；移除设备会撤销信任并触发空间密钥轮换。
- 可以清空本机历史或重新初始化配对安全状态。

## 日志与诊断

普通日志和诊断界面不得包含剪贴板正文、邀请秘密、密钥、完整摘要或完整网络帧。发布前运行 `scripts/release-safety-check.ps1` 检查被跟踪的秘密文件和发布目录中的调试产物。
